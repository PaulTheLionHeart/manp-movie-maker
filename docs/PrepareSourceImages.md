# Preparing Source Images

Before generating JPG animation frames, take a few moments to prepare the source PNG images.

Following these simple guidelines will help produce reliable movie animations.

---

## One Animation Per Directory

Keep each animation sequence in its own directory.

Avoid mixing PNG images from different animations in the same folder.

---

## Keep the Source Directory Pure

The source directory should contain only the PNG images belonging to the current animation.

Do not leave screenshots, title images, test images, or PNG files copied from other projects in the directory.

---

## Use Consistent Image Dimensions

All source PNG images should have identical width and height.

Mixing images with different dimensions may produce unexpected results in the generated animation.

---

## Before Generating JPG Frames

Before selecting **Prepare JPG Frames**, check that:

- the source directory contains only the intended PNG images;
- every PNG image has the same dimensions; and
- the images belong to a single animation sequence.

These simple checks help ensure reliable movie generation.
