# FFmpeg Installation

## Introduction

ManpMovieMaker uses FFmpeg to convert generated JPG frame sequences into high-quality MP4 movies.

FFmpeg is a free and widely used multimedia processing tool that supports a large range of video and audio formats.

FFmpeg is not included with ManpMovieMaker and must be installed separately.

---

## Downloading FFmpeg

Visit the official FFmpeg website:

https://ffmpeg.org

Download a current Windows build suitable for your system.

Installation packages and ZIP archives are both acceptable.

---

## Installing FFmpeg

Install or extract FFmpeg according to the instructions provided with the selected package.

After installation, verify that the FFmpeg executable is available on your computer.

---

## Verifying the Installation

Open a command prompt window and enter:

```text id="v0w4z6"
ffmpeg -version
```

If FFmpeg is installed correctly, version information similar to the following will be displayed:

```text id="r0v2az"
ffmpeg version ...
```

If Windows reports that FFmpeg is not recognised, refer to the next section.

---

## Adding FFmpeg to the System Path

Some FFmpeg installations automatically update the Windows PATH environment variable.

If the command:

```text id="k1j6nv"
ffmpeg -version
```

is not recognised, you may need to add the FFmpeg installation directory to the Windows PATH.

The exact procedure depends on the version of Windows being used.

After updating the PATH, open a new command prompt window and repeat the verification step.

---

## Using FFmpeg with ManpMovieMaker

The typical workflow is:

```text id="n4zt4w"
Render PNG images in ManpWIN
            ↓
Prepare Frames (S)
            ↓
Create FFmpeg Script (F)
            ↓
Copy FFmpeg Command
            ↓
Paste into Command Prompt
            ↓
Run FFmpeg
            ↓
MP4 Movie

For best compatibility with H.264 MP4 output, use even frame dimensions. Current versions of ManpMovieMaker can automatically crop odd frame dimensions by one pixel during JPG generation if required.

```

The FFmpeg command generated by ManpMovieMaker can be copied directly from the Create FFmpeg Script dialog and pasted into a command prompt window.

---

## Troubleshooting

### FFmpeg Is Not Recognised

If Windows reports:

```text id="v2h7lb"
'ffmpeg' is not recognised as an internal or external command
```

then either:

* FFmpeg is not installed.
* FFmpeg is not accessible via the system PATH.
* A new command prompt window is required after installation.

---

### The Movie Is Not Created

Verify that:

* FFmpeg is installed correctly.
* The JPG frame sequence exists.
* The generated FFmpeg command was copied correctly.
* The specified output folder is writable.
